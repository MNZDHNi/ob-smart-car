# 输出模式

1. 准双向口
2. 推挽输出（强推动，适合PWM）
3. 高阻输入
4. 开漏输出

# 寄存器（Px）

直接读写引脚电平状态

# 模式配置寄存器（8bit）

| PxM1 | PxM2 | mode |
| ---- | ---- | ---- |
| 0    | 0    | 双向   |
| 0    | 1    | 推挽   |
| 1    | 0    | 高阻   |
| 1    | 1    | 开漏   |

### 案例

`P2M0 = 0x00; P2M1 = 0x00;` // 将P2口所有引脚设为**准双向口**  
`P2M0 = 0xFF; P2M1 = 0x00;` // 将P2口所有引脚设为**推挽输出**  
`P2M0 = 0x01; P2M1 = 0x00;` // 仅将P2.0设为**推挽输出**，其余为准双向口
